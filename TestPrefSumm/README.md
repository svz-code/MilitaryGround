# TestPrefixSumm

Попробовал написать на Dlang свою (недо) реализацию префиксных сумм.  
Все вроде как должно быть верно, но это не точно  

## Компилирование и запуск приложения

Если вы хотите скомпилировать и сразу запустить:
`
dub run
`

Если вы хотите просто скомпилировать:
`
dub build
`

Скомпилированная версия есть в корне программы.

Исходный код (как и само приложение) Вы можете использовать как хотите. (Лицензия MIT)
